реализация grep c флагами `-e`, `-i`, `-v`, `-c`, `-l`, `-n`, `-h`, `-s`, `-f`, `-o` и их сочитаний(в тестировании ограничивался парными). За пример логики взыт GNU.
примечательно - первая проба архитектуры (3 назависимых друг от друга блока: ручной парсер, task, output; со строгим разделением ответственности).
замечания - сообщение "No such file or directory" выводится через stdout а на через stderr (ради незагромождения кода, и потому что по мне, у этого сообщения больше имформативная задача).
